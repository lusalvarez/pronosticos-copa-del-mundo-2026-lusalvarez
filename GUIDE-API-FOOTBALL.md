# 🌐 Guide : Obtenir une clé API gratuite pour récupérer les matchs

## 🎯 API recommandée : API-Football (RapidAPI)

**Avantages :**
- ✅ 100 requêtes gratuites par jour
- ✅ Données en temps réel
- ✅ Coupe du Monde 2026 incluse
- ✅ Facile à utiliser

---

## 📝 Étape 1 : Créer un compte RapidAPI

1. Allez sur : **https://rapidapi.com/api-sports/api/api-football**
2. Cliquez sur **"Sign Up"** (en haut à droite)
3. Créez un compte avec :
   - Votre email
   - Ou connectez-vous avec Google/GitHub

---

## 🔑 Étape 2 : Obtenir votre clé API

1. Une fois connecté, vous êtes sur la page de l'API-Football
2. Cliquez sur l'onglet **"Pricing"**
3. Sélectionnez le plan **"Basic (Free)"** - 100 requêtes/jour
4. Cliquez sur **"Subscribe"**
5. Votre clé API apparaît dans l'onglet **"Endpoints"**
6. Cherchez **"X-RapidAPI-Key"** dans les headers
7. **Copiez cette clé** (elle ressemble à : `abc123def456...`)

---

## 💾 Étape 3 : Configurer la clé dans l'application

1. Ouvrez votre application de pronostics
2. Allez dans l'**Écran administrateur**
3. Cliquez sur **"⚙️ Configurer l'API"**
4. Collez votre clé API
5. Cliquez sur **"Sauvegarder"**
6. ✅ Vous pouvez maintenant récupérer les matchs automatiquement !

---

## 🚀 Étape 4 : Récupérer les matchs

1. Dans l'écran administrateur
2. Cliquez sur **"🌐 Récupérer les matchs depuis Internet"**
3. Sélectionnez la compétition : **"Coupe du Monde 2026"**
4. Choisissez la phase : **"Phase de groupes"** ou **"Tous les matchs"**
5. Cliquez sur **"Récupérer"**
6. ✅ Les matchs sont importés automatiquement !

---

## 📊 Limites du plan gratuit

- **100 requêtes par jour**
- Suffisant pour :
  - ✅ Récupérer tous les matchs de la Coupe du Monde (1 requête)
  - ✅ Mettre à jour les résultats plusieurs fois par jour
  - ✅ Utilisation normale de l'application

---

## 🔄 Alternative : API-Football.org

Si vous préférez une autre API :

1. Allez sur : **https://www.api-football.com/**
2. Créez un compte gratuit
3. Obtenez votre clé API
4. Même processus de configuration

**Limites :** 100 requêtes par jour également

---

## ⚠️ Important

- **Ne partagez jamais votre clé API** publiquement
- La clé est stockée localement dans votre navigateur
- Si vous partagez l'application, chaque utilisateur doit avoir sa propre clé

---

## 💡 Conseils

1. **Testez votre clé** : L'application vérifie automatiquement si la clé fonctionne
2. **Économisez vos requêtes** : Récupérez les matchs une fois, puis mettez à jour manuellement
3. **Plan payant** : Si vous dépassez 100 requêtes/jour, passez au plan Pro (9$/mois)
